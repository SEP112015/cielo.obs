# 🌌 Cielo Obs

Aplicación móvil para registrar observaciones del cielo offline.

## 🚀 Funcionalidades

- Registro de observaciones
- Foto y audio
- GPS 
- Filtros
- Perfil del usuario
- Borrado seguro
- Funciona sin internet

## 🛠 Tecnologías

- Flutter
- SQLite
- image_picker
- geolocator
- record
- audioplayers

## 📦 Instalación

Descargar APK y ejecutar en Android.
